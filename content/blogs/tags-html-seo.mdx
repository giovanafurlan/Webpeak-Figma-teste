<h2>Tags HTML SEO</h2><p>Quer aparecer no topo do ranking dos principais buscadores? Então temos que trabalhar!</p><h2><strong>Como usar Meta Tags HTML</strong></h2><p>Adicionar <strong>meta tags</strong> em seu site irá te fazer subir ao topo automaticamente, certo? Errado.&nbsp;</p><p>As Meta tags são mais uma peça de um grande ‘quebra-cabeça algorítmico’ que os principais buscadores levam em consideração ao decidir quais os resultados são relevantes e, apresentados aos usuários que realizaram a pesquisa.</p><p>Embora ainda haja debate sobre quais <strong>Meta Tags</strong> permanecem úteis e importantes para os motores de busca.</p><p>Tenha em mente que as <strong>Meta Tags</strong>, definitivamente, não são uma solução mágica para melhor rankear nos principais buscadores da Internet (como Google, Bing, Yahoo). No entanto, elas ajudam a contar para os Buscadores e (consequentemente) usuários sobre o que se trata seu site, e quando implementadas/utilizadas de forma incorreta, o impacto negativo pode ser devastador.</p><blockquote><strong>Não deixe de ler:</strong> <a href="https://www.webpeak.com.br/blogs/mobile-marketing-conheca-e-aumente-suas-vendas" target="_blank" data-rt-link-type="external"><strong>Mobile marketing: Conheça e aumente suas vendas!</strong></a></blockquote><h3>Utilizando Heading Tag em SEO: H1, H2, H3, H4, H5 e H6</h3><p>Esses são recursos de programação HTML utilizados para destacar títulos e sub-títulos de uma página/conteúdo. Não é apenas uma questão estética, texto grande ou pequeno, e sim uma hierarquia do que está presente nas páginas do site. A letra 'H' é uma abreviação em inglês para&nbsp;<em><strong>Header</strong></em>, que significa <em>Cabeçalho</em>.&nbsp;</p><p>Conceitualmente o H1 é o cabeçalho mais importante, mais forte dentro de um conteúdo e com a fonte maior, além de ser o elemento que o Google utiliza para determinar qual o <strong>assunto principal</strong> que está sendo abordado.</p><p>É recomendado que você utilize apenas um H1 por página, sempre focando na palavra-chave principal que você deseja rankear. As outras Heading Tags, como H2 e H3 (entre as outras), podem ser utilizadas como sub-títulos em uma página, e pensando em <strong><a href="https://www.webpeak.com.br/ferramenta-de-seo-all-in-one" data-rt-link-type="external">SEO</a></strong>, são um excelente recurso para incluir a palavra-chave principal e termos complementares, por exemplo: <em>Características do Vestido Florido</em> e <em>Medidas do Vestido Florido</em> (em caso de produtos), <em>Faixas do Albúm OK Computer - Radiohead</em> (em caso de um blog de música).</p><figure class="w-richtext-figure-type-image w-richtext-align-floatleft" data-rt-type="image" data-rt-align="floatleft" data-rt-max-width="360px" style="max-width:360px"><div><img src="https://daks2k3a4ib2z.cloudfront.net/569f974a6fd517c6508cdb15/5874d8868956b2544765d5bc_tags-html-seo.gif" alt="Tags HTML SEO"></div></figure><h2><strong>O que são Meta Tags?</strong></h2><p><strong>Meta Tags HTML</strong> são marcações de dados em páginas de destino (home, produto, categoria, etc), e devem ser adicionadas entre as Tags de Head no código HTML do site.</p><p>Os textos dessas marcações não é exibido na página, mas sim no código dela e diz aos navegadores informações específicas sobre a página.&nbsp;</p><p>Simplesmente, "explica" a página para um navegador poder compreendê-lo.</p><p><strong>Exemplo de um código com meta tags:</strong></p><p><em>&lt;Head&gt;&nbsp;&lt;Title&gt; Título da página/nome do produto&lt;/ title&gt;&nbsp;&lt;Meta name = "description" content = "Boa Descrição Aqui"&gt;&nbsp;&lt;Meta http-equiv = "Content-Type" content = "text / html; charset = UTF-8"&gt;&nbsp;&lt;/ Head&gt;</em></p><h2><strong>Tag Title</strong></h2><p><em>&lt;Title&gt; Título da página/nome do produto&lt;/ title&gt;</em></p><p>São apresentados nas páginas de resultados do motor de busca (SERP), exibem trechos de visualização para uma determinada pesquisa e, <strong>são importantes tanto para <a href="https://www.webpeak.com.br/blogs/saiba-o-que-e-seo-e-como-vai-ser-positivo-para-seu-negocio" data-rt-link-type="external">SEO</a> quanto para compartilhamento social</strong>.</p><p>A Tag Title aparece no bloco de cabeçalho da página, deve conter a palavra-chave escolhida para ser trabalhada e não ultrapassar um limite de 70 caracteres.</p><blockquote><strong>Não deixe de ler:</strong> <a href="https://www.webpeak.com.br/blogs/como-aumentar-o-lucro-do-seu-e-commerce" data-rt-link-type="external" target="_blank"><strong>Saiba como aumentar o lucro do seu e-commerce</strong></a></blockquote><h3><strong>Por que Tag Title são importantes para SEO?</strong></h3><p>A Tag Title é o texto principal que descreve sua página online. Elementos como título tem sido considerados um dos elementos de <strong><a href="https://www.webpeak.com.br/blogs/estrategia-de-seo-onpage-e-link-building" data-rt-link-type="external">SEO On-page</a></strong> mais importantes, aparecem em três locais principais: navegadores, as páginas de resultados de pesquisas e sites externos.</p><blockquote>A Tag Title é a TAG HTML mais importante para SEO.</blockquote><h2><strong>Meta Tag Description</strong></h2><p>Complemento ao Titulo...</p><p><em>&lt;Meta name = "description" content = "Sua Descrição Aqui"&gt;</em></p><p>Esta é uma <strong>meta tag</strong> bastante útil pois ela simplesmente, explica aos motores e aos próprios usuários que estão pesquisa, sobre o assunto da página.</p><p>Uma descrição atraente pode motivar pesquisadores a clicar em seu site a partir da <a href="https://www.webpeak.com.br/blogs/segredos-de-analise-serp-e-analise-de-ranking-para-seo" data-rt-link-type="external"><strong>SERP</strong></a>, especialmente se a descrição inclui as palavras-chave que eles estavam procurando. Ela facilita o entendimento para o motor de busca.</p><p>Sua descrição <strong>não deve ter mais do que 170 caracteres (incluindo espaços)</strong>.</p><p>A Descrição contribui para o aumento da taxa de clique sem seu site: quando se escreve uma boa descrição, isso não só ajuda a manter sua posição no Google, mas também ajuda a obter mais pessoas clicando no seu site e com maior qualidade. Uma descrição bem escrita não só informa aos usuários o que está em sua página, mas também os atrai para visitar o seu site. É como boa fachada. <strong>Sites com descrições pobres tem menos cliques</strong> e os motores de busca irão reclassificá-lo em favor de outros sites.</p><p><strong>&gt; Descrições vazias</strong>: É possível uma descrição estar vazia? Sim. E quando isso acontece, o Google e o Bing o preenchem para você. Na verdade, às vezes você pode preferir a versão do buscador.</p><h2><strong>Meta Tag Keywords</strong></h2><p>O único motor de busca que olha para as <strong>Meta Tags de Palavras-chave</strong> é o &nbsp;<a href="https://www.webpeak.com.br/blogs/bing-webmaster" data-rt-link-type="external"><strong>Bing</strong></a> (da Microsoft) &nbsp;e usá-la para ajudar a detectar spam. Para evitar ferir seu site, a sua melhor opção é a de nunca adicionar essa tag.</p><p>Se isso é radical demais para você, pelo menos, certifique-se de não ter enchido 300 <a href="https://www.webpeak.com.br/principais-ferramentas/recursos-de-palavras-chave" data-rt-link-type="external"><strong>palavras-chave</strong></a> na esperança aparecer melhor rankeado.&nbsp;</p><p>Isso não vai funcionar. Desculpa.</p><p>Se você já tem Meta Tag Keywords em seu site, mas não são spam, não há razão para retirá-los urgentemente. Tudo bem deixá-los por enquanto, basta tirá-los quando tiver tempo, &nbsp;isso reduz o peso da página e <a href="https://www.webpeak.com.br/blogs/tempo-de-carregamento-do-site-dicas-de-como-melhorar-seu-site" data-rt-link-type="external"><strong>tempo de carregamento</strong></a> e melhorar sua qualidade on-page.</p><blockquote><strong>Não deixe de ler:</strong> <a href="https://www.webpeak.com.br/blogs/reenviar-sitemap-para-os-principais-buscadores" data-rt-link-type="external"><strong>Reenviar Sitemap Para os Principais Buscadores</strong></a></blockquote><h2><strong>Meta Tag Robots</strong></h2><p>Meta Tag Robots ainda é uma das marcas mais importantes. Permite especificar que uma determinada página&nbsp;<em>não</em>&nbsp;devem ser indexada por um motor de busca.</p><p>Ainda é comum que uma página seja desindexada porque alguém acidentalmente adicionou uma ‘tag noindex’ para o site. Entender essa tag é de vital importância.</p><p><strong>Aqui estão os quatro implementações da Meta Tag Robots Meta e o que significam:</strong></p><p>&lt;Meta name = "robots" content = "NOINDEX, nofollow"&gt;</p><p><strong>Significa:</strong> "Não indexe esta página. Não siga os links na página." Esta tag é frequentemente usada quando um site está em desenvolvimento. O desenvolvedor irá ‘habilitar a opção’ noindex / nofollow nas páginas do site, afim de mantê-las invisíveis para os motores de busca. Ao lançar o seu novo site, confira se a Tag foi removida.</p><p>&lt;Meta name = "robots" content = "INDEX, NOFOLLOW"&gt;</p><p><strong>Significa:</strong> "Do Índice desta página. Não siga os links na página." Sua página será Indexada mas seus links a outras páginas não será seguido. Isso vai quebrar o caminho link em seu site a partir desta página para outras páginas.</p><p>&lt;Meta name = "robots" content = "NOINDEX, follow"&gt;</p><p><strong>Significa:</strong> "Não Indexe esta página. Não siga os links na página." A página vai sair da Indexação, mas seus links para outras páginas serão seguidos. Isso não vai quebrar o caminho do link em seu site.</p><p>&lt;Meta name = "robots" content = "index, follow"&gt;</p><p><strong>Significa:</strong> "Do Índice desta página. Não siga os links na página." Sua página será indexada e seus links para outras páginas serão seguidos. Isso não quebra o caminho do link em seu site.</p><blockquote><strong>Leia também:</strong> <a href="https://www.webpeak.com.br/blogs/seo-porque-seu-conteudo-nao-esta-aparecendo-no-google" data-rt-link-type="external" target="_blank"><strong>SEO: Porque seu conteúdo não está aparecendo no Google</strong></a></blockquote><h2><strong>Tag Charset</strong></h2><p>Todos os sites devem validar Charset. Certifique-se se na sua página está entregando código &nbsp;HTML usando caracteres em inglês.</p><p><em>&lt;Meta http-equiv = "Content-Type" content = "text / html; charset = UTF-8"&gt;</em></p><h2><strong>Outras Meta Tags</strong></h2><p>Existem outras Meta Tags, mas nenhum é realmente considerado útil hoje em dia.
