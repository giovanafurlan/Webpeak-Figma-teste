<p>Sem dúvidas a <a href="https://www.webpeak.com.br/blogs/google-webmaster-tools-melhore-seu-site-com-ajuda-do-google"><strong>Webmaster tools da Search Console</strong></a> são ferramentas de análise e acompanhamento de usuários essencial para todo e qualquer site. Entenda por que toda empresa que possui&nbsp;um site deve utilizar essa ferramenta, para encontrar problemas e diversas informações vitais para o negócio online. Sua utilização diária e com frequência não irá consertar suas páginas, mas te dará uma excelente direção dos passos a serem tomados.</p><h2><strong>Para que serve Webmaster tools</strong></h2><p>O <strong>Google Search Console</strong> também é conhecido como <a href="https://www.webpeak.com.br/blogs/google-webmaster-tools-melhore-seu-site-com-ajuda-do-google"><strong>Google Webmaster Tools</strong></a>, são as mesmas funções de análise de comportamento dentro do site.</p><p>Google Search Console é uma ferramenta que permite compreender como o Google vê o seu site para fazer as otimizações necessárias.</p><p>A ferramenta surgiu em junho de 2005 com o objetivo de manter o Google “informado” das novas páginas criadas. &nbsp;</p><p>Hoje o <a href="https://www.webpeak.com.br/blogs/google-search-console-entenda-o-funcionamento"><strong>Search Console</strong></a> abrange diversas funcionalidades quanto a análises de cada site, dando um parecer detalhado sobre a indexação da página no Google.</p><blockquote>Leia também: <a href="https://www.webpeak.com.br/blogs/como-colocar-seu-e-commerce-no-topo-do-google-e-alavancar-vendas" target="_blank">Como colocar seu e-commerce no topo do Google e alavancar vendas</a></blockquote><h2><strong>Como configurar Webmaster tools</strong></h2><p>1. &nbsp;&nbsp;&nbsp;É necessário criar uma conta. Se tiver uma conta no Gmail, basta confirmar o usuário. Caso não tenha clique<a href="https://accounts.google.com/ServiceLogin/signinchooser?service=sitemaps&passive=1209600&continue=https%3A%2F%2Fwww.google.com%2Fwebmasters%2Ftools%2F&followup=https%3A%2F%2Fwww.google.com%2Fwebmasters%2Ftools%2F&flowName=GlifWebSignIn&flowEntry=ServiceLogin" target="_blank">&nbsp;aqui</a>.</p><p>2. &nbsp;&nbsp;&nbsp;Assim que entrar na página inicial clique no botão vermelho “Add a site” e adicione seu URL.</p><p>3. &nbsp;&nbsp;&nbsp;Na página seguinte clique na aba “Alternate methods” e selecione “<a href="https://www.webpeak.com.br/blogs/tags-html-seo"><strong>HTML tag</strong></a>”.</p><p>4. &nbsp;&nbsp;&nbsp;Em seguida o Google irá fornecer um <strong>código HTML</strong> para que você cole na home page do seu site.</p><p>5. &nbsp;&nbsp;&nbsp;Depois que colar o código na página e salvar as alterações, aguarde alguns instantes e retorne para o Search Console. Clique no botão “Verify”.</p><p>6. &nbsp;&nbsp;&nbsp;Agora é só colocar a mão na massa!</p><h3><strong>Quais as principais ferramentas do Google Search Console</strong></h3><h4>Aparência de pesquisa (Search Appearance)</h4><p>Na área Search Appearance, é possível analisar as otimizações que você pode realizar para melhorar a aparência da sua página quando aparecer nos <strong>resultados de busca do Google</strong>, uma vez que a ferramenta faz a leitura de títulos, descrições, imagens, etc.</p><h4>Tráfego de pesquisa (Search Traffic)</h4><p>Esta ferramenta é útil para mostrar o que as pessoas estão procurando no campo de busca. O relatório consegue apresentar o tráfego orgânico para a sua página, e tirar insights de possíveis otimizações.</p><p>No relatório você consegue ver todos os dados apresentados como:</p><p>1. &nbsp;&nbsp;&nbsp;Palavras-chave do seu site que apareceram nos resultados de busca;</p><p>2. &nbsp;&nbsp;&nbsp;Páginas do seu site que foram referenciadas;</p><p>3. &nbsp;&nbsp;&nbsp;Países que acessaram seu site;</p><p>4. &nbsp;&nbsp;&nbsp;Quais foram os acessos em aparelhos móveis e desktop;</p><p>5. &nbsp;&nbsp;&nbsp;Identifica quais foram as pesquisas para web ou imagem;</p><p>6. &nbsp;&nbsp;&nbsp;Análise por datas;</p><p>7. &nbsp;&nbsp;&nbsp;Quantidade de cliques;</p><p>8. &nbsp;&nbsp;&nbsp;Quantas visualizações o link teve para o seu site;</p><p>9. CTR (click-through-rate);</p><p>10. Posição média do seu site nos resultados de busca.</p><blockquote>Leia mais: <a href="https://www.webpeak.com.br/blogs/como-usar-a-webpeak-para-melhorar-os-resultados-de-seu-ecommerce" target="_blank">Como usar a WebPeak para melhorar os resultados de seu ecommerce</a></blockquote><h4>Índice do Google (Google Index)</h4><figure class="w-richtext-figure-type-image w-richtext-align-floatleft" style="max-width:40%" data-rt-type="image" data-rt-align="floatleft" data-rt-max-width="40%"><div><img src="https://uploads-ssl.webflow.com/5a4661af7b791a00019cfaee/5b5a2b67b4ef1e8082adf1a8_google-search-console-tudo-que-voce-precisa-saber-sobre-webmaster-tools.jpeg" alt="Google Search Console"></div></figure><p>O Google Index mostra as páginas <a href="https://www.webpeak.com.br/blogs/ferramentas-do-google-para-melhorar-sua-presenca-na-internet" target="_blank"><strong>indexadas pelo Google</strong></a>, como está o desempenho do site, e a identificação de palavras-chave. Assim como as páginas que foram bloqueadas e as que foram removidas.</p><p>Sempre fique atento as seguintes colocações:</p><p>1. &nbsp;&nbsp;&nbsp;Mudanças que demonstram problemas de segurança;</p><p>2. &nbsp;&nbsp;&nbsp;Quedas inesperadas, pois indicam servidor sobrecarregado ou dificuldade de acesso ao seu conteúdo;</p><p>3. &nbsp;&nbsp;&nbsp;Volume de índices altos, indicando conteúdos duplicados;</p><p>4. &nbsp;&nbsp;&nbsp;Aumento de páginas indexadas, uma vez que o Google passe a rastreá-las.</p><h4>Rastreamento (Crawl)</h4><p>Nesta parte, é possível observar <a href="https://www.webpeak.com.br/blogs/entenda-como-funciona-o-rastreamento-google"><strong>como o Google rastreia seu site</strong></a> para indexar, e quais os problemas encontrados.&nbsp;</p><p>Se existem dificuldades para os robôs vasculharem as páginas do seu site na busca de conteúdos relevantes, você conseguirá identificar.</p><blockquote>Saiba também: <a href="https://www.webpeak.com.br/blogs/entenda-as-diferencas-entre-sitemap-e-robots" target="_blank">Entenda as diferenças entre Sitemap e Robots</a></blockquote><h4>Problemas de segurança</h4><p>Se o seu site tiver sido invadido ou infectado, você ficará sabendo nessa área.
