Aliás, pense um pouco nessa questão: como comunicar o conteúdo de uma imagem para um robô?<br></p><p>Por exemplo, imagine que um crawler vai ler o código <a href="https://www.webpeak.com.br/blogs/veja-como-melhorar-seo-sabendo-tudo-sobre-html">HTML</a> do seu novo conteúdo, que fala sobre os diferentes tipos de saias disponíveis no mercado.<br></p><p>Você seleciona uma bela imagem de uma saia longa para ilustrar o trecho que explica esse tipo de saia para o seu leitor. Porém, o nome do arquivo é “IMG438.png”. O que o crawler terá acesso é isso:<br></p><ul><li>&lt;img src="”IMG438.png”/"&gt;<br></li></ul><p>Isso já te mostra que não é uma informação muito útil para explicar o<strong> conteúdo da imagem</strong> que você utilizou no conteúdo. Ele é útil, pois ajuda pessoas com deficiência visual a compreender o conteúdo, mas também são exibidos quando o navegador do usuário não consegue carregar o arquivo da imagem por algum motivo.<br></p><p>E ainda mais importante é que ele é, também, mais um dos fatores que o Google utiliza para tentar entender uma imagem e sua relevância para o usuário.<br></p><p>Na hora de escrever o conteúdo do atributo alt, é importante descrever o que está na imagem de forma objetiva e breve, preferencialmente usando a palavra-chave que queremos <a href="https://www.webpeak.com.br/blogs/como-otimizar-seo-on-page-e-melhorar-o-rank">rankear</a> (tome muito cuidado com os excessos). O motivo é que o algoritmo do Google utiliza o conteúdo dos parágrafos perto da imagem como contexto para compreender o conteúdo dela.<br></p><p>Nesse sentido, se você estiver falando sobre “saias longas para festas”, por exemplo, é interessante colocar uma foto de uma utilizando saia longa para contextualizar.</p><h3>URL da imagem</h3><p>É muito importante estruturar a URL do seu site para que você tenha um bom rankeamento no <a href="https://www.webpeak.com.br/noticias/google-x-huawei-o-que-acontece-depois-da-suspensao-de-uso-do-android">Google</a>. A lógica é a mesma quando falamos de imagens.<br>O importante é organizar a URL dos seus arquivos de mídia de forma lógica e concisa, para comunicar de modo rápido ao buscador que aquilo é uma imagem e qual seu conteúdo.<br></p><blockquote>Leia mais: <a href="https://www.webpeak.com.br/blogs/seo-101-nocoes-basicas-de-search-engine-optimization">SEO 101: Noções básicas de Search Engine Optimization</a> </blockquote><h3>Dimensões da imagem</h3><p>Realize um teste nesse momento: abra uma nova aba no seu navegador e procure por alguma imagem no Google.
