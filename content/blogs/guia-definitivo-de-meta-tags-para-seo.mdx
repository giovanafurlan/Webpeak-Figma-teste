<p>Olá, o que é SEO você provavelmente já sabe, não é mesmo? </p><p>Para esclarecer melhor qualquer dúvida: <a href="https://www.webpeak.com.br/blogs/search-engine-optimization-para-ser-o-primeiro-no-google"><strong>Search Engine Optimization</strong></a><strong> (SEO)</strong>, que em português significa <strong>Otimização para Buscadores</strong>, faz referência a otimização de conteúdo para que ele apareça mais facilmente nas buscas feitas por usuários. </p><p>Uma <strong>metadescrição </strong>trata-se de um pequeno texto que aparece abaixo do título da página nos resultados das buscas, e serve como um resumo do que o usuário encontrará ao clicar naquele link, funciona como um elemento de atração.</p><p>Uma vez que esse item defina se alguém visitará ou não o seu site, é importante que ele seja elaborado de maneira correta.</p><p>Para atender a um <a href="https://www.webpeak.com.br/blogs/seo-tutorial-como-alcancar-o-topo-dos-buscadores"><strong>SEO otimizado</strong></a>, acrescente a palavra-chave na área, como forma de aumentar a relevância.</p><figure class="w-richtext-figure-type-image w-richtext-align-floatleft" style="max-width:20%" data-rt-type="image" data-rt-align="floatleft" data-rt-max-width="20%"><div><img src="https://uploads-ssl.webflow.com/5a4661af7b791a00019cfaee/5d03cc467813df6e028fb49a_guia-definitivo-de-meta-tags-para-seo%202.jpg" alt="meta tags para seo" width="auto" height="auto" loading="auto"></div></figure><p>Entretanto, não deixe de criar uma chamada criativa e que atue de modo a persuadir. Utilize palavras de impacto de modo a incentivar o usuário a clicar na página. Outra recomendação importante é que você mantenha o tamanho dentro de parâmetros ideais. Desse modo, ficará visível, sem textos cortados de forma abrupta e você irá manter o máximo de resultados.</p><p>Mas vamos falar melhor das <strong>Meta Tags</strong> <strong>para SEO</strong> agora mesmo!</p><blockquote>Leia também: <a href="https://www.webpeak.com.br/blogs/5-maneiras-de-provar-o-valor-de-seo-para-seus-clientes">5 maneiras de provar o valor de SEO para seus clientes </a></blockquote><h2>O que são Meta Tags para SEO </h2><figure class="w-richtext-figure-type-image w-richtext-align-fullwidth" style="max-width:1317px" data-rt-type="image" data-rt-align="fullwidth" data-rt-max-width="1317px"><div><img src="https://uploads-ssl.webflow.com/5a4661af7b791a00019cfaee/5cb8df3b5ed5f1c4011daaa7_meta%20tags.jpg" alt="meta tags para seo" width="auto" height="auto" loading="auto"></div></figure><p><strong>Meta Tag</strong> é um comando implementado no código de páginas web, dentro da área Head do site (entre as tags &lt;head&gt; e &lt;/head&gt;) para repassar instruções a programas externos ou ações mais simples, como informar qual a pessoa responsável pelo desenvolvimento da página. </p><p>Determinadas <strong>Meta Tags</strong> são utilizadas para passar aos <a href="https://www.webpeak.com.br/blogs/existe-vida-alem-do-google-desvendamos-os-mecanismos-de-busca"><strong>mecanismos de busca</strong></a> como o Google, Bing, entre outros, instruções sobre o título da página e uma descrição breve a ser exibida nos resultados de busca, quais páginas devem ou não ser indexadas, assim como outras instruções.</p><p>Em resumo,<strong> Meta tags</strong> são comandos em HTML que descrevem o conteúdo de uma página para os mecanismos de busca, ou seja, fornecem dados como título ou breve resumo sobre o assunto abordado para programas externos. Por meio das<strong> meta tags</strong> é possível indicar a autoria do responsável pelo desenvolvimento da página. Elas normalmente não são visíveis aos usuários, mas estão em que ele realizam suas pesquisas.</p><h3>Importância de Meta Tags para SEO</h3><p><strong>Meta Tags</strong> tratam-se de uma importante ferramenta de comunicação entre o webmaster e os sites de busca, como vamos apresentar mais abaixo. Para muitas pessoas, porém, SEO é somente isso, <strong>Meta Tags ou </strong><a href="https://www.webpeak.com.br/blogs/tags-html-seo"><strong>Tags HTML SEO</strong></a>. Isso está longe de ser verdade. O Google utiliza perto de 250 variáveis para determinar os resultados de pesquisa, e as meta tags são apenas algumas delas. Diante disso, atente-se a outros meios. </p><h3>Como visualizar Meta Tags</h3><p><strong>Meta Tags </strong>não são exibidas no tela principal do Browser, ou seja, no conteúdo exibido ao usuário. O jeito mais fácil de visualizá-las é pela exibição do código fonte da página. Cada Browser tem a sua forma de fazê-lo. <br>Normalmente, basta clicar com o botão direito em alguma área da página e clicar em "Exibir código fonte da página" ou algum texto similar a esse. <br>No Chrome, teclar Ctrl-U tem o mesmo efeito. Exibido o código-fonte, faça uma busca por &lt;meta ou &lt;title (geralmente digitando Ctrl-F) e veja como os meta tags da página estão especificados:</p><p>Há plugins que permitem visualizar rapidamente os meta tags de uma página. Um de nossos favoritos é o plugin gratuito Quirk Search Status para Firefox. </p><blockquote>Não deixe de ler: <a href="https://www.webpeak.com.br/blogs/a-webpeak-e-a-ferramenta-mais-completa-de-marketing-digital-do-mercado">A WebPeak é a ferramenta mais completa de marketing digital do mercado </a></blockquote><h2>Meta Tags mais importantes em SEO</h2><h3>Meta Title</h3><p>O campo Title não é uma<strong> Meta Tag</strong>, porém é o mais importante comando em uma página web para <a href="https://www.webpeak.com.br/blogs/voce-um-seo-ninja-com-webpeak"><strong>SEO</strong></a>.<br></p><p>Exemplo de um Title:&lt;title&gt;Project management software, online collaboration: Basecamp&lt;/title&gt;</p><h3>Meta Description (meta name="description")</h3><p>É a <strong>Meta Description</strong> que normalmente define a descrição exibida nos resultados do Google, ou qualquer outro site de busca. <br>Descreva o conteúdo da página de uma forma a estimular o interesse das pessoas em conhecer mais sobre o conteúdo da página, para aumentar a visitação do seu site. </p><p>Exemplo de um <strong>Meta Description</strong>: &lt;meta name="description" content="Veja no Terra as últimas notícias e as melhores coberturas ao vivo do Brasil e do Mundo, Esportes, Diversão, Vida e Estilo, assista os melhores vídeos no TerraTV e ouça as melhores músicas no Sonora." /&gt;</p><h3>Meta Keywords (meta name="keywords")</h3><p>No passado, há muitos anos atrás, os algoritmos dos sites de busca tinham certa dificuldade em interpretar a relevância de sites para determinadas palavras-chave e utilizavam o conteúdo da <strong>Meta Keyword </strong>para ajudá-los a interpretar o conteúdo das páginas. </p><p>De forma rápida, os webmasters perceberam que bastava incluir uma infinidade de palavras nessa tag para aparecer nas buscas, mesmo para palavras não relacionadas ao conteúdo da página. </p><p>A <a href="https://www.webpeak.com.br/blogs/o-que-a-experiencia-do-usuario-tem-a-ver-com-o-sucesso-do-seu-e-commerce"><strong>experiência do usuário</strong></a> era ameaçada ao ser direcionada a uma página que no final das contas não possuía o conteúdo buscado. Os sites de busca passaram por uma evolução e hoje desconsideram a <strong>meta keyword</strong>. </p><p>Uma vez que a meta keyword não é exibida ao usuário em qualquer momento, seja nos resultados de busca ou no conteúdo do site, não há de fato qualquer razão lógica para que o Google a considere como um fator de relevância.</p><p>As meta keywords são utilizadas para pesquisar as palavras-chave de concorrentes desavisados. Exemplo de um meta keyword: &lt;meta name="keywords" content="climática, previsão climática, desenvolvimento, tempo, clima,"/&gt;</p><blockquote>Não deixe de ler: <a href="https://www.webpeak.com.br/blogs/por-que-meu-site-nao-aparece-como-resultado-de-pesquisas">Por que meu site não aparece como resultado de pesquisas </a></blockquote><h3>Meta Robots (meta name="robots")<br></h3><figure class="w-richtext-figure-type-image w-richtext-align-floatright" style="max-width:30%" data-rt-type="image" data-rt-align="floatright" data-rt-max-width="30%"><div><img src="https://uploads-ssl.webflow.com/5a4661af7b791a00019cfaee/5d03cc74fdc2c6209744cb7c_guia-definitivo-de-meta-tags-para-seo%203.jpg" alt="meta tags para seo" width="auto" height="auto" loading="auto"></div></figure><p><strong>Meta Robots </strong>é uma meta tag criada para passar orientações aos <strong>robots dos buscadores</strong>. Cada página deve ter o seu meta robots. Veja alguns dos parâmetros que podem ser passados:</p><ul><li>index: indexe esta página - exiba-a em seus resultados de busca;</li><li>noindex: não indexe esta página - não a exiba nos resultados de busca. Útil para páginas como de login e acesso à intranet;</li><li>follow: siga os links desta pagina para descobrir novas páginas (reveja Googlebot, robots);</li><li>nofollow: nenhum dos links desta página deve ser seguido;</li><li>nosnippet: orienta o site de busca a não exibir a descrição da página nos resultados de busca;</li><li>noodp: orienta o Google não utilizar a descrição do diretório DMOZ em seus resultados (snippet);</li><li>noarchive: instrui o Google a não exibir a versão em cache da página;</li><li>noimageindex: não indexe nehuma imagem da página.<br><br></li></ul><p>Veja alguns exemplos de <strong>Meta Robots</strong>:</p><ul><li>&lt;meta name="robots" content="index, follow"&gt; - sintaxe mais comum de meta robots - orienta os buscadores a indexar o conteúdo da página e seguir todos os links para descobrir páginas novas.<br><br></li><li>&lt;meta name="robots" content="noindex, nofollow"&gt; - orienta os buscadores a não indexar o conteúdo da página e impede-a de seguir os links para descobrir novas páginas</li></ul><p><strong>Nota:</strong> o comando para orientar ao Google a não indexar uma página pode também ser dado do arquivo <a href="https://www.webpeak.com.br/blogs/entenda-as-diferencas-entre-sitemap-e-robots"><strong>robots.txt</strong></a>.</p><p>Você leu um conteúdo completo sobre <strong>Meta Tags </strong>para SEO!
