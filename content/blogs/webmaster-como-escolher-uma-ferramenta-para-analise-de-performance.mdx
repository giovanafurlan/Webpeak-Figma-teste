ele&nbsp;busca o link duas vezes, uma com um <em>user agent</em> para dispositivo móvel e outra vez com um <em>user agent</em> para computador.</p><p>Uma outra opção também bem conhecida atualmente são plugins específicos para um CMS, como por exemplo o<a href="https://wordpress.org/plugins/p3-profiler/" target="_blank">&nbsp;P3 - Plugin Performance Profiler</a>,&nbsp;desenvolvido pelo GoDaddy.</p><p>A ferramenta é bem interessante, já que muitas vezes a lentidão no carregamento do site pode ser por conta de plugins mal configurados ou pelo grande quantidade&nbsp;deles que estão instalados.</p><p>Uma <strong>FERRAMENTA PARA ANÁLISE DE PERFORMANCE</strong> que tem sido bastante comentada, é o <a href="https://website.grader.com/" target="_blank">Website Grader da Hubspot</a>, ele não somente faz a análise da performance do seu site como também de responsividade, SEO e segurança. Essa <strong>FERRAMENTA PARA ANÁLISE DE PERFORMANCE&nbsp;</strong>consegue oferecer um relatório geral do que pode ser melhorado tecnicamente em seu site.</p><h3><strong>#2 Ferramenta para analise de performance:</strong> Facilidade de uso</h3><p>A ferramenta que você precisa escolher deve ser bem intuitiva e que funcione bem - caso contrário a tendência é que você acabe deixando-a de lado após alguns dias e assim deixar a performance do site para depois.</p><p>Avalie com cuidado a ferramenta para análise de performance que você vai escolher.
