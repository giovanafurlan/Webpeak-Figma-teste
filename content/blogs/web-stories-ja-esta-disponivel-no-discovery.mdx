<p>Cada mergulho é um flash, não é mesmo? Depois dos stories presente no Facebook, Instagram e WhatsApp, agora o Web Stories também está disponível no Discovery do Google.&nbsp;</p><p>O carrossel do Discovery é acessível, por ser um aplicativo do Google para Android e iOS. No momento que você tem ativado no seu celular, é possível consultar as notícias de última hora e também tem acesso ao Web Stories.&nbsp;</p><p>Para acompanhar as novidades do Web Stories é simples. Ao tocar em uma Story no carrossel, o leitor entra em uma experiência envolvente em tela cheia. Você também consegue avançar para a próxima página ou deslizar para uma história totalmente diferente na sequência.</p><p>O carrossel no Discovery foi uma grande sacada da empresa, em que o Google sugere mais Web Stories nos resultados da Busca, baseado em suas pesquisas recentes, nos dispositivos móveis em todo o mundo.</p><p>Isso é inteligência artificial, minha gente!</p><p>Para ajudar a impulsionar o ecossistema das Web Stories, o Google trabalhou durante este ano com um grupo diverso de publicações no Brasil, EUA e Índia para colaborar no produto e seus recursos.</p><p>E se você faz parte dos empreendedores digitais fique sabendo que, os autores têm o controle total sobre monetização, hospedagem, compartilhamento e inclusão de links em seus conteúdos. É um resultado garantido.&nbsp;</p><p>Com mais de 800 milhões de usuários todos os meses, as Web Stories se tornaram mais uma forma de criar conteúdo relevante para a sua audiência.&nbsp;</p><h3>Mas, como fazer Web Stories no Discovery?</h3><p>Qualquer pessoa, ou negócio, pode utilizar a ferramenta de Web Stories. Mais de 2.000 sites já fizeram publicação de Stories indexados pelo Google, e começaram a gerar resultados consistentes.&nbsp;</p><p>Hoje existem ferramentas que facilitam o processo de criação, com botões para links, CTA’s, e formatos já pensados para SEO. Algumas das plataformas de edição mais utilizadas são: Web Story,&nbsp; MakeStories e NewsroomAl.</p><h3><a href="https://wp.stories.google/">Web Story</a> - Plugin para WordPress</h3><p>Uma ferramenta simples de manusear, o famoso arrasta e solta, plugin feito especialmente para quem utiliza o Wordpress. Com ele você consegue...</p><ul><li><strong>Definir um fundo de página</strong>: arrastar imagens ou vídeos até a borda da página para definir como plano de fundo da página;&nbsp;</li><li><strong>Editar mídia</strong>: você pode clicar duas vezes sobre a página ou vídeo para redimensionar, centralizar ou cortar;</li><li><strong>Sobreposição de Fundo</strong>: assim que definir o fundo da página, é possível adicionar uma sobreposição de gradiente sólida, linear ou radeal de acordo com a sua identidade visual;</li><li><strong>Zona segura</strong>: para facilitar o processo de criação a ferramenta disponibiliza a zona segura, assim os elementos principais ficarão ajustados da forma correta;</li><li><strong>Histórias visuais</strong>: é possível incorporar histórias em suas postagens de blog, é só abrir o bloco e selecionar o Web Stories e inserir o link.&nbsp;&nbsp;</li></ul><h3>NewsroomAl</h3><p>Essa é outra ferramenta que segue o mesmo processo do Web Story, mas neste caso, está relacionado a uma plataforma de edição, sem estar vinculado a nenhuma plataforma de conteúdo.&nbsp;</p><p>A interface da plataforma é bem intuitiva com planos que vão do gratuito ao corporativo. Você pode fazer aquele teste antes de contratar a ferramenta.&nbsp;</p><h3><a href="https://makestories.io/">MakeStories</a></h3><p>Mais uma ferramenta muito interessante e fácil de mexer. Ela fornece algumas opções como:&nbsp;</p><h4>Abordagem de arrastar / soltar / clicar</h4><p>Construa suas histórias da web simplesmente arrastando e soltando elementos e organize várias propriedades CSS e de animação com nossa abordagem de um clique.</p><h4>Zero Coding Garantido</h4><p>MakeStories é um editor 100% WYSWIG para Web Stories. Você nunca terá que codificar uma única linha, pois cada aspecto de sua história é controlável.</p><h4>Rich Stock Library</h4><p>A plataforma possui biblioteca de imagens, ícones, ilustrações, formas, molduras e gifs livres de royalties à sua disposição para acelerar a construção de sua história.</p><h2>Porque fazer Web Stories no Discovery</h2><p>Para começar as Web Stories permitem que você exiba vídeo, fotos e áudio em tela cheia de maneira integrada, isso melhora a experiência do usuário. Sem contar que a pessoa que tem em mãos um dispositivo móvel pode ter acesso ao conteúdo.</p><p>Outro ponto interessante de quem faz esse tipo de divulgação é que você também pode criar um link para outro conteúdo. Em outras palavras, mais tráfego orgânico para as suas páginas na internet, e até mesmo novos seguidores nas redes sociais. O que acha?</p><p>Não&nbsp; existe segredo, tudo você consegue gerenciar dentro do seu próprio site, mas não se esqueça que para ter resultado o conteúdo precisa ser atrativo e dentro dos padrões SEO.</p><h3>Dicas para um bom conteúdo</h3><p>Se você quer que seus Web Stories faça sucesso, então é necessário seguir algumas dicas.&nbsp;</p><ol><li><strong>Fique no topo</strong>: tanto quanto possível, use vídeo envolvente em tela inteira na proporção de retrato (9:16) para uma experiência envolvente;</li><li><strong>Seja objetivo</strong>: evite muito texto, escreva apenas o essencial, variando o tamanho do tipo e estilos.&nbsp;</li><li><strong>Garanta legibilidade</strong>: se for colocar texto, nada de deixar de um jeito difícil de ser compreendido. O texto precisa ser fácil de ler com contraste e separação adequada.</li><li><strong>Atenção as animações</strong>: coloque animações nos objetos separadamente para garantir um visual mais atrativo.</li><li><strong>Não fuja da sua identidade visual</strong>: é muito importante manter sua identidade visual em evidência, para que as pessoas associem a sua marca.</li><li><strong>Otimize para o Google</strong>: certifique-se de que sua história possa ser descoberta usando a ferramenta Web Story para visualizá-la.</li></ol><p>Viu como é fácil? Cada vez mais empresas estão migrando para o digital, e nesse universo tão concorrido é necessário utilizar todas as formas possíveis de divulgação para atrair mais pessoas para sua solução.&nbsp;</p><p><br></p>
