<figure class="w-richtext-figure-type-image w-richtext-align-floatleft" style="max-width:30%" data-rt-type="image" data-rt-align="floatleft" data-rt-max-width="30%"><div><img src="https://uploads-ssl.webflow.com/5a4661af7b791a00019cfaee/5d389f70e9a7e789779808f2_aposto-que-voce-ainda-nao-sabe-fazer-otimizacao-de-imagens%20(2).gif" alt="Otimização de imagens " width="auto" height="auto" loading="auto"></div></figure><p>Na linguagem do <strong>marketing digital</strong>, a <strong>otimização de imagens</strong> é essencial para proporcionar uma experiência de navegação agradável para o usuário. Além disso, irá garantir que o <strong>“robô” do Google</strong> consiga identificar esse tipo de informação.<br></p><p>Para fazer bom uso das imagens em sua <a href="https://www.webpeak.com.br/blogs/estrategia-de-seo-onpage-e-link-building">estratégia de <strong>SEO</strong></a>, não basta somente tirar as melhores fotos e fazer a melhor edição. Você deve também <strong>otimizar as imagens</strong>, para que então elas possam ser lidas pelos robôs do Google e tenham boas posições nos resultados de busca.<br></p><p>Neste conteúdo, você aprenderá algumas dicas de <strong>SEO para Imagens</strong>.<strong> </strong>Veja como fazer a <strong>otimização de suas imagens</strong> de maneira correta.<br></p><blockquote>Leia também: <a href="https://www.webpeak.com.br/blogs/a-gente-criou-um-tutorial-para-voce-criar-uma-persona-ainda-hoje">A gente criou um tutorial para você criar uma persona ainda hoje!</a></blockquote><h2>Entenda a importância da otimização de imagens</h2><p>Vamos começar falando do <strong>Google Imagens</strong>, que é um recurso do buscador que permite aos usuários que encontrem imagens com mais facilidade e disponibilizá-las para que o usuário encontre-as nas buscas.<br></p><p>Diante disso, para que uma imagem fique nos<a href="https://www.webpeak.com.br/blogs/como-colocar-seu-e-commerce-no-topo-do-google-e-alavancar-vendas"> primeiros resultados</a>, ela precisa ser relevante aos usuários, e isso envolve de estar&nbsp; otimizada para o<strong> algoritmo do Google Imagens</strong>.<br></p><p>Ao pesquisarmos por alguma palavra-chave que tenha relação com imagens, como uma capa para celular, por exemplo, o <strong>Google </strong>já irá disponibilizar, inúmeras vezes na primeira colocação dos resultados orgânicos, as imagens que podem ser mais relevantes frente à busca do usuário.<br></p><p>Vamos supor que você possua um <a href="https://www.webpeak.com.br/blogs/como-fazer-link-building-para-e-commerce">e-commerce </a>que, entre outras coisas, vende porta-retratos. Nesse caso, é fundamental que suas imagens apareçam nos resultados da busca, afinal, os próprios usuários podem acessar seu site através das fotos que acharem mais interessantes.<br></p><p>Ter um conteúdo sobre porta-retrato que não possua imagens pode não ser muito relevante, e identificamos isso tanto para os usuários, quanto para o algoritmo do Google. Sem dúvida isso se aplica a vários outros tipos de negócio e termos.</p><h2>Como fazer a otimização de imagens para o Google</h2><p>Bom, agora você já compreende a importância de fazer <strong>otimização de imagens </strong>em seu site ou blog, a questão central que fica é: como fazer esse trabalho? Quais as melhores práticas para garantir um<a href="https://www.webpeak.com.br/blogs/melhorar-resultados-acompanhando-rank-de-palavras-chave"> bom ranqueamento</a> das suas imagens?<br></p><h3>Nome do arquivo da imagem</h3><p>Para o ranqueamento de imagens, o primeiro elemento se inicia ainda no seu computador, e é anterior ao <strong>upload da imagem </strong>para seus servidores: o nome do arquivo das mesmas.<br></p><p><strong>Otimização de imagens</strong>, ou seja, para otimizar suas imagens para o Google, o nome do arquivo da imagem é essencial, para compreender textos e a relevância de certos conteúdos usando um algoritmo e dando valor às palavras-chaves.<br></p><p>Entretanto, ainda estão aprendendo a ler imagens. Aliás, pense um pouco nessa questão: como comunicar o conteúdo de uma imagem para um robô?<br></p><p>Por exemplo, imagine que um crawler vai ler o<strong> código HTML</strong> do seu novo conteúdo, que fala sobre os diferentes tipos de saias disponíveis no mercado.<br></p><p>Você seleciona uma bela imagem de uma saia longa para ilustrar o trecho que explica esse tipo de saia para o seu leitor. Porém, o nome do arquivo é “IMG438.png”. O que o crawler terá acesso é isso:<br></p><ul><li>&lt;img src="”IMG438.png”/"&gt;<br></li></ul><p>Isso já te mostra que não é uma informação muito útil para explicar o conteúdo da imagem que você utilizou no conteúdo. Nesse sentido, veja como ficaria o código se o arquivo fosse melhor nomeado:<br></p><ul><li>&lt;img src="”saia_longa_preta_reta.png”/"&gt;</li></ul><p>Mesmo sem ver a imagem, conseguimos imaginar como ela é com um nome desses, não é mesmo?<br></p><p>Dessa forma, você facilita o entendimento para os algoritmos.&nbsp;<br></p><blockquote>Não deixe de ler: <a href="https://www.webpeak.com.br/blogs/content-experience-o-novo-content-marketing">Content Experience: o novo content marketing&nbsp;</a></blockquote><h3>Legenda da imagem</h3><p>Algo que talvez você não saiba é que as legendas embaixo de imagens são lidas, em média, 300% mais do que o conteúdo do texto. Dá pra acreditar?&nbsp;<br></p><p>Isso significa que as legendas são fundamentais por gerarem oportunidades para engajar seus leitores. Mas também o <strong>Google</strong>.<br></p><p>O <a href="https://www.webpeak.com.br/blogs/existe-vida-alem-do-google-desvendamos-os-mecanismos-de-busca">próprio buscador</a> admite que faz uso das legendas para extrair informações e tentar compreender a imagem.&nbsp;<br></p><p>Mas calma, isso não significa que você deva inserir legendas em 100% das imagens nos seus conteúdos, porém deve utilizar esse recurso sempre que tentar <strong>rankear </strong>um arquivo visual em específico.<br></p><p>O segredinho WebPeak para escrever uma boa legenda está em inserir informações que:</p><ul><li>ajudem a compreender o <strong>conteúdo da imagem</strong>;</li><li>tenham a <strong>palavra-chave que queremos rankear</strong>;</li><li>complementem o conteúdo do artigo.<br></li></ul><h3>Alt text</h3><p>Todas (eu disse t-o-d-a-s)&nbsp; as imagens na Internet possuem um atributo ALT, que serve para descrever o conteúdo delas.<br></p><p>Esse é o fator mais importante para fazer <strong>otimização de imagens </strong>de forma assertiva.<br></p><p>O<strong> alt text</strong> atua de modo a descrever o conteúdo da imagem em questão. Ele é útil, pois ajuda pessoas com deficiência visual a <a href="https://www.webpeak.com.br/blogs/devemos-produzir-conteudo-para-pessoas-ou-motores-de-busca">compreender o conteúdo</a>, mas também são exibidos quando o navegador do usuário não consegue carregar o arquivo da imagem por algum motivo.<br></p><p>E ainda mais importante é que ele é, também, mais um dos fatores que o <strong>Google</strong> utiliza para tentar entender uma imagem e sua<a href="https://www.webpeak.com.br/blogs/mobile-marketing-como-melhorar-a-experiencia-do-usuario"> relevância para o usuário</a>.<br></p><p>Na hora de escrever o conteúdo do atributo alt, é importante descrever o que está na imagem de forma objetiva e breve, preferencialmente usando a palavra-chave que queremos rankear (tome muito cuidado com os excessos). Veja um exemplo:<br></p><ul><li>ruim: &lt;img src="“saia_longa_de_cetim_preta.png”" alt="“saia" longa="" de="" cetim="" preta="" melhores="" marcas="" preços="" imperdíveis”=""&gt;</li><li>bom: &lt;img src="“saia_longa_de_cetim_preta.png”" alt="“saia" longa="" de="" cetim="" preta="" para="" festas”=""&gt;</li></ul><h3>Contexto da imagem</h3><p>De acordo com o Google, um dos elementos essenciais de <strong>otimização de imagens </strong>é o contexto em que a imagem é utilizada.<br></p><p>É importante que a figura seja adicionada ao seu artigo na parte em que ela é mais relevante. O motivo é que o algoritmo do Google utiliza o conteúdo dos parágrafos perto da imagem como contexto para compreender o conteúdo dela.<br></p><p>Nesse sentido, se você estiver falando sobre “saias longas para festas”, por exemplo, é interessante colocar uma foto de uma utilizando saia longa para contextualizar.</p><h3>URL da imagem</h3><p>É muito importante estruturar a URL do seu site para que você tenha um <strong>bom rankeamento no Google</strong>. Vira e mexe, <a href="https://www.webpeak.com.br/blogs/seo-101-nocoes-basicas-de-search-engine-optimization">nós da WebPeak</a>, dizemos isso. A lógica é a mesma quando falamos de imagens.<br><br>O importante é organizar a URL dos seus arquivos de mídia de forma lógica e concisa, para comunicar de modo rápido ao buscador que aquilo é uma imagem e qual seu conteúdo.<br></p><blockquote>Leia mais: <a href="https://www.webpeak.com.br/blogs/seo-101-nocoes-basicas-de-search-engine-optimization">SEO 101: Noções básicas de Search Engine Optimization</a>&nbsp;</blockquote><h3>Dimensões da imagem</h3><p>Realize um teste nesse momento: abra uma nova aba no seu navegador e procure por alguma<strong> imagem no Google.</strong> Pode ser, por exemplo, “filhotes de gatos”.<br></p><p>Se você prestou atenção, deve ter reparado que a grande maioria das imagens mostradas possuem mais ou menos as mesmas dimensões.<br>O <a href="https://www.webpeak.com.br/blogs/saiba-como-aumentar-o-trafego-do-seu-blog-com-o-google-imagens">Google tende a favorecer imagens </a>mais ou menos retangulares, em proporções como 16:9 ou 4:3. O que faz com que dimensões como 1366 x 768 (16:9) ou 720 x 480 (4:3) tenham um rankeamento melhor do que imagens totalmente fora do padrão (como alguma coisa muito horizontal ou muito vertical).<br></p><p>Esteja atento às dimensões da imagem para garantir excelência quando o assunto for <a href="https://www.webpeak.com.br/blogs/sabe-fazer-seo-imagens-aprenda-tudo-aqui">SEO Imagens</a>.&nbsp;</p><h3>Compartilhamentos da imagem</h3><p>Algo curioso é que o que aumenta o ranqueamento de uma imagem é a quantidade de vezes que essa mídia foi reproduzida em outros sites via Embed.</p><p>Embed trata-se de uma<a href="https://www.webpeak.com.br/blogs/tags-html-seo"> tag HTML</a> que permite que um arquivo de mídia (foto, música, vídeo) de um site seja incorporada em outra página. Quanto mais isso acontece com uma imagem, mais o <strong>Google </strong>a enxerga como relevante e a ranqueia melhor.<br></p><p>Esta informação é extremamente útil quando elaboramos infográficos e outras imagens ricas em conteúdo. Uma boa ideia é facilitar e incentivar o embed dessas mídias em outras páginas para melhorar o ranqueamento delas no <strong>Google Imagens</strong>.</p><h3>Posicionamento da imagem na página</h3><p>Já mencionamos que a<strong> posição da imagem no artigo</strong> em que ela está inserida é fundamental para o seu ranqueamento no Google. Isso ocorre porque o conteúdo dos parágrafos próximos da figura ajudam a fornecer contexto ao buscador.<br></p><p>Entretanto, existe outro elemento de posicionamento que garante melhor ou pior ranqueamento para uma imagem: se ela está perto do topo da página ou não.<br></p><p>É recomendado pelo Google que a principal imagem do seu artigo, aquela que melhor complementa seu conteúdo, deve ficar perto do topo da sua página. Dessa forma, ela ganhará o maior impacto no ranqueamento e fortalecerá a <a href="https://www.webpeak.com.br/blogs/ferramenta-de-palavra-chave-para-sugestoes-matadoras">palavra-chave</a> que você almeja destacar.<br></p><blockquote>Leia também:<a href="https://www.webpeak.com.br/blogs/investir-em-redes-sociais-esse-e-o-momento"> Investir em redes sociais? Esse é o momento!</a></blockquote><h3>Dados estruturados</h3><p>Outro ponto muito importante é que o <strong>Google Imagens </strong>usa o sistema de<strong> dados estruturados</strong> para criar certos “selos de destaque” para otimizar a busca dos seus usuários.<br></p><p>Como funciona:<br></p><p>Se você produz um artigo que seja de um dos<strong> temas compatíveis com os selos do Google</strong>, pode usar os dados estruturados em suas imagens. Assim, elas ganharão o selo e indicarão ao usuário que é exatamente o que ele procura.<br></p><p>O Google Imagens é compatível com 3 tipos de dados estruturados:</p><ul><li><a href="https://www.webpeak.com.br/blogs/aplicativos-para-videos-crie-sua-estrategia-de-marketing-do-seu-proprio-celular">vídeos</a>;</li><li>receitas;</li><li>produtos.</li></ul><p>Se sua imagem tiver relação com um desses assuntos, você pode usar os<strong> dados estruturados </strong>para ganhar o selo de destaque e ver sua imagem rankear melhor no Google Imagens.</p><h3>Sitemap de imagem</h3><p><strong>Sitemap </strong>é uma ferramenta essencial para o SEO da sua página, afinal, a partir dela que é indicado ao crawler do Google a hierarquia dentro do seu site e como ler o seu conteúdo de forma prioritária.<br></p><p>É bem provável que você tenha um desses no seu blog ou site. Mas você sabia que pode criar um sitemap exclusivo para suas imagens?&nbsp;<br></p><p>Quem recomenda isso é o próprio Google. O buscador incentiva e <a href="https://support.google.com/webmasters/answer/178636">ensina como criar um sitemap de imagens</a> para facilitar o trabalho do crawler de achar seus arquivos de mídia e, claro, garantir um melhor ranqueamento para seu site. Não deixe de realizar esta tarefa.&nbsp;</p><h3>Formato da imagem</h3><p>Não é todo formato de imagem que consegue ser captado pelo Google Imagens. Diante disso, use o formato correto se quiser ser ranqueado adequadamente.<br></p><p>Os aceitos são os seguintes:</p><ul><li>JPEG;</li><li>PNG;</li><li>MBP;</li><li>GIF;</li><li>WebP;</li><li>SVG;</li><li>imagens in-line.</li></ul><p>O formato mais utilizado é o JPEG, pois é aquele com maior capacidade de compressão e consegue entregar arquivos com tamanhos pequenos. Entretanto, como ponto negativo, ele não trabalha com transparências, como o GIF e PNG.<br></p><p>Está preparado para fazer <strong>otimização de imagens</strong> de forma assertiva?&nbsp;<br></p><p>Coloque essas informações em prática e veja o Google consagrar seu negócio!&nbsp;<br></p><p>Até a próxima! :)&nbsp;<br></p>
